# Buyer Profiles Directory

This directory contains individual profiles for each potential buyer identified during the M&A process.

## Purpose
Maintain detailed intelligence on each buyer to support:
- Targeted outreach and messaging
- Meeting preparation
- Negotiation strategy
- Buyer selection decisions

## Profile Structure

Each buyer profile should follow this template:

```markdown
# [Buyer Name]

## Basic Information
- **Type:** Strategic / Financial
- **Industry:** [Industry]
- **Geography:** [Location]
- **Size:** [Revenue/AUM]
- **Website:** [URL]

## Strategic Fit
- Why would they want this target?
- What synergies exist?
- How does it fit their strategy?

## M&A Activity
- Recent acquisitions
- Typical deal size
- Integration track record
- Valuation multiples paid

## Financial Capacity
- Balance sheet strength
- Access to capital
- Fund size (if PE)
- Recent fundraising

## Key Contacts
| Name | Title | Email | Phone | LinkedIn | Notes |
|------|-------|-------|-------|----------|-------|
| | | | | | |

## Engagement History
| Date | Activity | Participants | Outcome | Next Steps |
|------|----------|--------------|---------|------------|
| | | | | |

## Buyer Motivation
- Primary drivers for acquisition
- Specific needs this target addresses
- Urgency level
- Budget/authority

## Likely Concerns
- What issues will they focus on?
- What might cause them to walk away?
- What questions will they ask?

## Negotiation Strategy
- Leverage points with this buyer
- Likely negotiation style
- Key decision makers
- Timeline expectations

## Competitive Intelligence
- Other deals they're pursuing
- Other options they have
- Competitive position vs. other buyers

## Cultural Fit
- Management style
- Company culture
- Employee treatment history
- Integration approach

## Scoring
- **Strategic Fit:** [1-10]
- **Financial Capacity:** [1-10]
- **Likelihood:** [1-10]
- **Cultural Fit:** [1-10]
- **Overall Score:** [1-10]

## Status
- [ ] Identified
- [ ] Researched
- [ ] Contacted
- [ ] NDA Signed
- [ ] CIM Distributed
- [ ] Management Meeting
- [ ] LOI Received
- [ ] Selected for DD
- [ ] Closed

## Notes
[Free-form notes and observations]

## Documents
- Links to related documents
- Correspondence
- Presentations
```

## Naming Convention

Files should be named: `{buyer-name-slug}.md`

Examples:
- `techcorp-ag.md`
- `globalinvest-pe.md`
- `industry-leader-gmbh.md`

## Update Frequency

- Update after each interaction
- Update when new intelligence is gathered
- Mark status changes immediately
- Track engagement progression

## Integration

These profiles are used by:
- **Buyer Relationship Manager** (primary owner)
- **Market Intelligence** (research and updates)
- **Managing Director** (strategic decisions)
- **Document Generator** (buyer-specific materials)
